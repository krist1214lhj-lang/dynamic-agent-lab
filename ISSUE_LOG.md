# ISSUE_LOG

## 목적
Vercel에 배포된 `dynamic-agent-lab` 앱의 실제 사용 테스트 중 발견되는 오류, UI 불편사항, 결과 품질 문제, 개선 아이디어를 기록한다.

## 현재 배포 주소
https://dynamic-agent-lab.vercel.app

## 기록 규칙
- 문제를 발견하면 날짜, 테스트 조건, 기대 결과, 실제 결과를 기록한다.
- 원인 추정은 확실하지 않으면 "추정"이라고 표시한다.
- 수정 완료 후 상태를 "완료"로 변경한다.
- 수정 후 smoke test 결과를 함께 기록한다.
